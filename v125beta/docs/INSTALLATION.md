# Installation

## المتطلبات
- PHP 8.0+ (مفضل 8.1+)
- MySQL/MariaDB
- صلاحية كتابة لمجلد `storage/` (cache/logs/ratelimit)

## خطوات سريعة
1) ارفع الملفات إلى الاستضافة.
2) اضبط ملف البيئة `.env` (أو الإعدادات المعتمدة في مشروعك).
3) تأكد من:
   - `storage/cache` قابل للكتابة
   - `storage/logs` قابل للكتابة
4) افتح الموقع وتأكد من عمل الصفحات العامة.
5) للدخول للوحة التحكم:
   - `/admin/login.php`

## ملاحظات استضافة
- لو كان الموقع داخل مجلد فرعي (subfolder) تأكد أن `base_url()` مضبوط.
- على shared hosting، File cache هو الافتراضي (لا يحتاج Redis).
