# Security Policy (Godyar CMS)

## الإبلاغ عن ثغرات
- لا تنشر تفاصيل الثغرة علنًا قبل إصلاحها.
- أرفق وصفًا وخطوات إعادة الإنتاج ونسخة النظام وبيئة التشغيل.

## الاستجابة للطوارئ
1) اعزل الموقع مؤقتًا
2) بدّل الأسرار (DB/SMTP/API)
3) اجمع السجلات (storage/logs + server logs)
4) افحص مجلدات الرفع والإضافات
5) حدّث إلى أحدث إصدار
